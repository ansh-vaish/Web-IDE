# Next.js WebContainer Starter

A minimal, WebContainer-safe Next.js starter template.

## Important
This template is configured for **WebContainers** and uses:

```bash
next dev --webpack
```

Do **not** switch back to plain `next dev` in WebContainers, because that may try Turbopack and fail.

## Setup

```bash
npm install
npm run dev
```

## Included
- Next.js 14.2.15
- React 18.3.1
- App Router
- TypeScript

## Notes
This starter is intentionally minimal to maximize compatibility in browser-based IDEs.
